# Ubuntu 24.04 构建修复

## 问题

在 Ubuntu 24.04 (Noble) 中，`libtinfo5` 包已被移除，导致依赖安装失败。

## 解决方案

已从依赖列表中移除 `libtinfo5`，并添加 `libtinfo6` 作为替代（如果需要）。

## 修复内容

```yaml
# 之前（会失败）
libtinfo5

# 现在（已修复）
# libtinfo5 已移除
# 如果需要，会自动安装 libtinfo6
```

## 验证

构建应该能够成功安装所有系统依赖。

如果还有其他依赖问题，请检查：
1. 包名是否正确
2. 包是否在 Ubuntu 24.04 中可用
3. 是否有替代包
