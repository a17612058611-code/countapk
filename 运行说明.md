# 如何运行应用

## 🚀 快速运行（推荐）

### 方法1：使用运行脚本（最简单）

```bash
./run_app.sh
```

这个脚本会自动：
- 检查并激活虚拟环境
- 安装依赖（如果需要）
- 启动应用

### 方法2：手动运行

```bash
# 1. 激活虚拟环境
source venv/bin/activate

# 2. 运行应用
python main.py
```

## 📱 应用功能

运行后，你会看到一个窗口，包含：

1. **日期显示** - 显示当前日期
2. **分数输入** - 输入0-100的分数
3. **描述输入** - 输入分数描述
4. **保存按钮** - 保存当天的分数
5. **统计信息**：
   - 今天分数和描述
   - 总分
   - 平均分
6. **查看历史记录** - 查看所有历史记录

## 🔧 如果遇到问题

### 问题：ModuleNotFoundError: No module named 'kivy'

**解决**：
```bash
# 确保激活了虚拟环境
source venv/bin/activate

# 安装依赖
pip install -r requirements.txt
```

### 问题：窗口没有出现

**解决**：
- 检查是否有错误信息
- 确保在图形界面环境中运行
- 尝试使用 `./run_app.sh` 脚本

### 问题：中文显示为乱码

**解决**：
- 应用已配置中文字体支持
- 如果仍有问题，检查系统字体

## 📝 使用示例

1. **输入分数**：在"今天分数"输入框中输入 `85`
2. **输入描述**：在"分数描述"输入框中输入 `今天表现不错`
3. **保存**：点击"保存"按钮
4. **查看结果**：界面会自动更新显示统计信息
5. **查看历史**：点击"查看历史记录"按钮查看所有记录

## 🎯 数据存储

所有数据保存在 `scores.json` 文件中，即使关闭应用也会保留。
